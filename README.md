# Sudoku-Solver

A web application which solves 9 x 9 sudoku puzzle.

## Tech stack used
  1) Vanilla Javascript
  2) Cascading Style Sheets
  3) HTML

## How is it working?
  Implemented using BACKTRACKING algorithm.
